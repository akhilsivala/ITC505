Stylistic Choices in CSS

1. Font Choices:

  • I opted for the 'Arial' font family as it is a clean and widely supported sans-serif font, providing a professional and modern appearance.
  • For the header, I used a different font color and background color to create a visually distinct and attention-grabbing section, enhancing readability.

2. Heading and Text Sizes:

  • The main text size is set to 1.6 times the default size, ensuring comfortable reading without overwhelming the user.
  • The <h1> tag is used for the title, as it represents the highest-level heading on the page, aligning with semantic HTML practices. This choice helps in creating a clear document structure.

3. Color Choices:

  • The background color of the body is a light shade (#f4f4f4), providing a subtle contrast with the content and improving overall readability.
  • The header has a contrasting background color (#3498db) to distinguish it from the rest of the content, drawing attention to important information.

4. Margin and Padding Choices:

  • Margin and padding for the body and articles are set to provide consistent spacing and a balanced layout, enhancing the visual appeal of the content.
  • Specific padding is applied to the header for a more aesthetically pleasing appearance.

HTML Abuse:

  1. The title is marked with an <h1> tag as it is the highest-level heading on the page. This aligns with proper HTML semantics and improves accessibility.
